# Практична робота №9–10

## Тема:
Асинхронна робота з даними у JavaScript: fetch(), JSON, рендеринг каталогу та керування станом інтерфейсу

---

## Мета роботи

Метою роботи є створення data-driven вебпроєкту на основі існуючого HTML/CSS сайту з використанням JavaScript.  
Контент завантажується з JSON-файлу, динамічно відображається в інтерфейсі та підтримує взаємодію з користувачем.

---

## Реалізований функціонал

У проєкті реалізовано наступні можливості:

- асинхронне завантаження даних через fetch()
- використання async/await та обробка помилок через try/catch
- динамічний рендеринг карток на основі JSON-даних
- стан завантаження ("Завантаження...")
- стан помилки ("Помилка завантаження даних")
- стан порожнього результату ("Нічого не знайдено")

---

## Робота з даними

Дані зберігаються у файлі:


data/items.json


Кожен об'єкт містить:
- id
- title
- description
- category
- price
- rating

Дані завантажуються через fetch() та обробляються у JavaScript.

---

## Функціонал каталогу

### Пошук
- реалізовано пошук у реальному часі
- пошук виконується по title та description
- результати оновлюються без перезавантаження сторінки

### Фільтрація
- фільтр за категоріями
- працює разом із пошуком

### Сортування
- сортування за ціною
- змінює порядок відображення елементів

---

## Пагінація

- реалізована кнопка "Показати ще"
- спочатку відображається 4 елементи
- при натисканні підвантажуються нові
- кнопка зникає, коли всі елементи показані

---

## Обране (Favorites)

- додавання елементів в обране
- збереження в localStorage
- стан відновлюється після перезавантаження сторінки
- можливість видалення з обраного

---

## Деталі елемента

- реалізовано модальне вікно
- відображається детальна інформація про елемент
- дані підтягуються з JSON
- модальне вікно можна закрити

---

## Додатковий функціонал

- перемикання теми (dark/light)
- збереження теми в localStorage
- форма з валідацією
- збереження введених даних форми
- автоматичне визначення активної сторінки
- кнопка "вгору"
